## Detailed Description

The OrderTriggerReport represents the successful placement of an order and is passed in RCallbacks::TriggerReport().

The OrderTriggerReport class conveys information about the new order.
**See also**
  REngine::modifyOrder     REngine::sendOrder     REngine::sendOrderList
